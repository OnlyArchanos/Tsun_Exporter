# Tsun Exporter

Tsun Exporter is a Chrome extension that allows you to easily extract your manga reading list from WeebCentral and export it to MyAnimeList (MAL) or MangaUpdates.

## Features

- Extract manga lists directly from WeebCentral profiles or library pages.
- Verify missing metadata automatically using the MangaUpdates API.
- Export to MyAnimeList XML format for direct import.
- Export to MangaUpdates text format.
- Fully backgrounded exports that run independently of the popup.

## Installation

1. Download or clone this repository as a .zip and extract it.
2. Open your Chromium browser and go to `chrome://extensions/`.
3. Toggle "Developer mode" ON (top right corner).
4. Click "Load unpacked" and select the extracted folder.

## Usage

1. Navigate to a WeebCentral profile
2. Click the Tsun Exporter extension icon.
3. Click "Extract from Current Tab".
4. Once extracted, switch to the "Export" tab.
5. Select either "MyAnimeList XML" or "MangaUpdates Links" to begin the background generation process.
